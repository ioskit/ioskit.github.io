Add your CVs here!

You can add your own CVs on this website in order to take advantage of the list of iOS Features, Apple and External Frameworks provided here: your CVs will become dynamic by showing witch ones you used.

Your CVs can be "public" (visible in the list above) or not (only people having the URL can read it... and those going to the Github repository). You can create both a public and a private CVs. For example, the public CVs of John DOE is automatically listed on this page, but not his private one: if you want to see the private CVs of John DOE you had to know its URL here.

To add your CVs here you have to:

  1. Get a local copy of the Github repo https://github.com/ioskit/ioskit.github.io
  2. Duplicate the file /_cvs/template.md and name it like that: firstnamelastname.md or firstnamelastname-private.md.
  3. Edit this file
  4. Commit and create a Pull Request
